# n8n Workflows

This directory contains example n8n automation workflows for Sik.

Workflows use the Sik API for SERP scraping and common automation tasks.

## Usage

1. Import any `.json` workflow file into n8n.
2. Update HTTP Request node URLs to point to your Sik instance.
3. Adjust variables as needed.
